# Bert4Keras-LZ

代码说明
====================  
本代码将BERT、albert、Roberta预训练模型整合到wikisql与nl2sql表格问答数据集中，检验预训练模型对于表格问答的效果
以下是对代码运行的说明。


环境配置
================  
  tensorflow==1.14  
  keras==2.2.4  
  bert4keras==0.7.0 
  keras_bert==0.80 
  python==2.7/3.5  
>>


数据集下载
================  
  WikiSQL：https://github.com/salesforce/WikiSQL  
  NL2SQL:None


文件运行
===============  
  python wikisql_bert.py  
  python nl2sql_bert.py
